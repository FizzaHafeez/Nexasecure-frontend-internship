## Features
- Sticky navigation bar with smooth scrolling
- Hero section with intro, role, and call-to-action buttons
- Skills section with pill-style tags
- Project cards linking to Week 1 GitHub projects
- Contact section with email, GitHub, and LinkedIn links
- Fully responsive (desktop, tablet, mobile) with media queries
- Reduced-motion support for accessibility

## Tech Stack
- HTML5 (semantic structure)
- CSS3 (Flexbox, custom properties, media queries)
