## Demo Video Script (5–7 minutes)

**0:00–0:30 — Intro**
"Hi, I'm Fizza Hafeez, a frontend development intern at NexaSecure Tech. This is my Week 2 demo covering JavaScript and DOM manipulation projects."

**0:30–1:30 — Day 8: JS Basics**
Open the console, scroll through the output. Point out: variables and data types, arrow functions, the grade condition logic, and the string-vs-number debugging example at the end.

**1:30–2:30 — Day 9: DOM Part 1**
Click each button: change text, insert HTML, random color box, grow/reset, highlight list, add paragraphs. Say one line about `getElementById` vs `querySelectorAll`.

**2:30–3:30 — Day 10: DOM Part 2 (Events)**
Demo the counter (+/−/reset with color change), live text mirror, hover zone, press keys, drag the font slider. Mention everything uses `addEventListener`.

**3:30–5:00 — Days 11–12: Calculator**
Do a basic calculation, then a chained one (2 + 3 + 4). Show percent and square root. Trigger the divide-by-zero error. Type a calculation with the keyboard to show keyboard support.

**5:00–6:30 — Days 13–14: To-Do App**
Add 3–4 tasks, complete one, delete one, switch filters. Then the key moment: **refresh the page** and show that tasks persist. Explain in one sentence: "Tasks are saved as JSON in localStorage and reloaded on page load."

**6:30–7:00 — Outro**
"All code is on my GitHub with commits for each day. Next week: APIs and UI enhancement. Thanks for watching."

**Recording tips:** use OBS Studio (free) or Windows Game Bar (Win+G). Record in one take per section, keep the console visible for Day 8.
