# NexaSecure Tech Frontend Internship — Week 1

HTML & CSS Foundations (Days 1–7)

## Folder Structure
```
nexasecure-week1/
├── day01-html-basics/       # Semantic HTML: headings, paragraphs, lists, links
├── day02-forms-media/       # Forms, inputs, buttons, images, video, audio
├── day03-css-basics/        # Box model, selectors, colors, fonts (commented CSS)
```

## Daily Submission Checklist

| Day | Deliverable | Action |
|-----|-------------|--------|
| 1 | HTML code examples | Commit `day01-html-basics/` |
| 2 | Form demo | Commit `day02-forms-media/` + screenshot of the form |
| 3 | Commented CSS | Commit `day03-css-basics/` |
| 4 | Flexbox project | Commit `day04-flexbox/` |
| 5 | Landing page | Commit `day05-landing-page/` + screenshots (desktop & mobile) |
